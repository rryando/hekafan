# hekafan
A MEVN Stack (Mongo, Express, Vue.js, Node.js) boilerplate
## Setup Development Machine
1. Clone the repo `https://github.com/rryando/hekafan.git`

2. `cd hekafan`

3. Open client
```
cd client
yarn
npm run dev
```

4. Open server
```
cd server
yarn
npm start
```

5. Open `http://localhost:8080` in browser
